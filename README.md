# Airport Slot Reallocation under Congestion

This repository contains the GAMS optimization model and Python scripts accompanying the paper:

**"A Flight Rescheduling Model at a Schedule-Coordinated Airport to Incorporate Airlines' Preferences."**

## Repository Structure

```
src/
    GAMS optimization model
    Python scripts

data/
    Sample input datasets

docs/
    Documentation
```

## Data

The repository includes sample input datasets for demonstration purposes. They illustrate the required input format but are not the datasets used in the paper.

## Requirements

- GAMS 24.1.2
- Python 3.8.10
- pandas

# Data

This directory contains sample input files that follow the structure expected by the GAMS and Python codes. Users may replace these sample files with their own datasets while preserving the same format.
The sample data were created for demonstration purposes only and do not correspond to the datasets used in the paper of 'A flight rescheduling model at a schedule-coordinated airport to incorporate airlines' preferences'.

The sample datasets use the following specifications:
- Number of flights in each set (`i`, `j`, and `k`): **20**
- Number of time periods (`t`): **30**

Please note that the sample data is artificial and the code can face error when run. The purpose is just for understanding the structure of the data.
